Test Sensor Data

Send test sensor data to verify your system is working correctly

Instructions
  1. Register a Sensor: First, go to the admin panel and register a sensor to get a sensor key
  2. Enter Credentials: Use the sensor ID and key from the admin panel
  3. Send Data: Click "Send Test Data" to send sensor readings to the system
  4. Check Dashboard: Go to the main dashboard to see the data appear in real-time
  5. Test Alerts: Use "Test Alerts" to send data that will trigger alert conditions
Tip: Use the "Generate Random Data" button to simulate realistic sensor readings, or "Test Alerts" to trigger warning/critical alerts.
API Information

This form sends data to: POST /api/sensor/data

You can also send data programmatically using this JSON format:

{
  "sensor_id": "test_sensor_001",
  "sensor_key": "your_sensor_key_here",
  "hive_id": "test_hive_001",
  "temperature": 25.5,
  "humidity": 65.2,
  "gas_level": 45.0,
  "hive_weight": 12.5,
  "battery_level": 85,
  "signal_strength": 75
}
