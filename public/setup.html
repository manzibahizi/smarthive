Sensor API Information

API Endpoint
POST http://localhost:8000/api/sensor/data
Required Fields
  • sensor_id - Your sensor ID
  • sensor_key - Authentication key
  • hive_id - Hive identifier
  • temperature - Temperature in Â°C
  • humidity - Humidity percentage
  • gas_level - Gas level in ppm
  • hive_weight - Weight in kg
Example JSON
{
  "sensor_id": "sensor_001",
  "sensor_key": "your_key_here",
  "hive_id": "hive_001",
  "temperature": 25.5,
  "humidity": 65.2,
  "gas_level": 45.0,
  "hive_weight": 12.5,
  "battery_level": 85,
  "signal_strength": 75
}
